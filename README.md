# Spring Card

A small interactive greeting card for March 8th. Open the envelope, scroll through the message and photo galleries, then tap the heart at the end.

- Single-page HTML/CSS/JS
- Animated envelope, floating flowers, background music
- Slide navigation (click or swipe) after opening the envelope
- Responsive layout

Run locally with any static server (e.g. `npx serve .`) or open via [GitHub Pages](https://pages.github.com/).
